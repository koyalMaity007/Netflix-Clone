# Netflix Clone 🎬

A Netflix-inspired movie browsing web application built using HTML, CSS, JavaScript, Node.js, and Express.

This project fetches movie data from the TMDB API and displays:
- Popular movies
- Trending movies
- Top-rated movies
- Continue Watching section
- Movie trailers
- Search functionality

---

## 🚀 Live Demo

[Click Here](https://netflix-clone-sage-ten.vercel.app/)

---

## ✨ Features

- Dynamic movie fetching using TMDB API
- Hero banner section
- Horizontal scrolling movie rows
- Movie trailer modal popup
- Continue Watching using LocalStorage
- My List section
- Responsive Netflix-style UI
- Backend API protection using Express server

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
```

Install backend dependencies:

```bash
cd server
npm install
```

Start the backend server:

```bash
node index.js
```

Open `Netflix.html` in your browser.
