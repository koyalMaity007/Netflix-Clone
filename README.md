# Netflix Clone 🎬

A Netflix-inspired movie browsing web application built using HTML, CSS, JavaScript, Node.js, and Express.

This project fetches movie data from the TMDB API and displays:
- Popular movies
- Trending movies
- Top-rated movies
- Continue Watching section
- Movie trailers
- Search functionality

---

# Features

- Dynamic movie fetching using TMDB API
- Hero banner section
- Horizontal scrolling movie rows
- Movie trailer modal popup
- Continue Watching using LocalStorage
- My List section
- Responsive Netflix-style UI
- Backend API protection using Express server
